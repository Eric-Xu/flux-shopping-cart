React/Flux Shopping Cart
=========

Tutorial source: [Egghead.io]

Run Instructions
-----------

Check that node and npm are installed:
```sh
node -v
npm -v
```

Install Gulp.js:
```sh
npm install gulp -g
npm install gulp
```

Install additional Node modules:
```sh
npm install gulp-browserify gulp-concat react reactify es6-promise
```

Run Gulp build:
```sh
gulp

gulp watch
```
